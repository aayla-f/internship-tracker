# About

This is a simple web application for tracking internship applications.
Users can add, edit, delete, and view their applications. All data is stored in the browser using localStorage.

No backend or database is required.

# Description

The project helps students keep track of where they have applied and the status of each application.

It includes:
1. A main page to view applications
2. A form to add or edit applications
3. An analytics page to see application status statistics

The project runs fully in the browser.

## Files

This project contains five main files:

1. index.html – Shows the list of applications
2. add.html – Form to add or edit applications
3. analytics.html – Displays charts and analytics
4. script.js – Handles data and logic
5. style.css - Styling

## How to Run

1. Download or clone the project files.
2. Open index.html in a web browser.
3. The application will start automatically.

## Features

1. Add internship applications
2. Edit existing applications
3. Delete applications
4. View status analytics
5. Save data using localStorage
